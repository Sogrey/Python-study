Python的数据类型、变量

Python的数据类型

基本数据类型

  • 整数（int） eg: 8
  • 浮点数（float） eg: 8.8
  • 字符串（str） eg: "Python"
  • 布尔值（bool） eg: True False

判断数据类型

type(8)  # class 'int',type函数判断数据类型
type("python")  # class 'str'

数据类型转换

 int('8')  # 8 , "8"  -> 8
 str(123)  # '123' 
 bool(8)  # True

变量

   a        =         123
# ---      ---       -----
#变量名   为变量赋值   变量的值
  b = a # 变量a的值赋给变量b
