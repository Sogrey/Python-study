序列

序列是指一组有序数据并且可以通过下标偏移量访问它的成员。

  • 字符串 eg: "abcd"
  • 列表 eg: [0,'1',2,'3']
  • 元组 eg: ("abc","DEF")
chinese_zodiac = '鼠牛虎兔龙蛇马羊猴鸡狗猪'
print(chinese_zodiac[6]) # '马'
print(chinese_zodiac[0:4]) # '鼠牛虎兔'
print(chinese_zodiac[-1]) # '猪' ,倒序
